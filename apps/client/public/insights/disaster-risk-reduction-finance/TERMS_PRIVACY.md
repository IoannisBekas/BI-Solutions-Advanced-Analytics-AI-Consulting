## Privacy

The static public GitHub Pages dashboard does not require user accounts and does not intentionally collect personal data. Premium access, authentication, billing, and API use are handled through protected backend services for eligible users.

## Premium Data

Restricted, licensed, manually curated, or paid-only data must not be committed into the public static site. Premium data is served only through authenticated access after subscription, entitlement, and license checks.
