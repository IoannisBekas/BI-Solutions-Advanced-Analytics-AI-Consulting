## Public Beta

This dashboard is an independent public beta. It is provided for research, screening, and exploratory analysis. It should not be used as the sole basis for funding, procurement, insurance, emergency response, or policy decisions without verifying the underlying source records.

## Data Sources

The public dashboard uses open public datasets and links back to source publishers where possible. Source publishers retain their own licenses, terms, and update schedules. Users should consult the original source before republication or operational use.

## No Official Endorsement

References to public institutions, UN agencies, development banks, donors, or data publishers do not imply endorsement of this dashboard.

## Privacy

The static public GitHub Pages dashboard does not require user accounts and does not intentionally collect personal data. If premium features are activated later, authentication, billing, and API access must be handled by a separate backend with its own privacy controls.

## Premium Data

Restricted, licensed, manually curated, or paid-only data must not be committed into the public static site. Premium data should be served only through an authenticated backend after subscription, entitlement, and license checks.
