# Enhancement Roadmap

This roadmap tracks planned enhancements for the Country-Level DRR Risk and Finance Briefing after public release.

## Data Depth

- Add IATI activity and transaction ledgers with duplicate-detection rules.
- Add national budget lines, disaster management funds, climate budget tagging, and public investment program evidence where documents are public.
- Add private and blended finance signals from project documents, climate funds, insurer/reinsurer disclosures, and philanthropic reports.
- Add confidence labels for domestic, private, and mainstreamed DRR evidence layers.

## Premium Access

- Deploy the `premium-api` backend.
- Configure Stripe annual subscription products for Pro and Institution plans.
- Add authentication, entitlement checks, billing success flows, and billing cancellation flows.
- Keep restricted premium data outside GitHub Pages and serve it only through protected API access.

## User Workflows

- Save country profiles.
- Export country briefs as PDF.
- Add donor drilldowns and project lists with source links.
- Add organization accounts, API keys, export audit trails, team-level usage limits, and permissions.

## Data Governance

- Add source licensing notes.
- Add a data retention policy for future accounts.
- Add a country correction workflow.
- Add source freshness thresholds and anomaly detection for large year-to-year value changes.
- Add methodology version numbers to exports and reproducible release tags for public data snapshots.
